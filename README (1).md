## 🚀 Proyectos incluidos

### 1. Análisis de Personal — RRHH
Análisis exploratorio de un dataset de 207 empleados utilizando Python y Pandas. Incluye limpieza de datos, EDA y dashboard interactivo en Power BI.

**Tecnologías:** Python · Pandas · NumPy · Power BI · Excel

### 2. Analizador de Acciones Bursátiles
App web de análisis técnico bursátil publicada en Streamlit. Descarga datos de Yahoo Finance, calcula indicadores técnicos (SMA 20/50, RSI 14) y detecta señales de Golden Cross y Death Cross. Envía reportes por email automáticamente.

**Tecnologías:** Python · Pandas · Matplotlib · yfinance · Streamlit

🔗 [Probar app en vivo](https://stock-analyzer-jjdyycgbtlpqjckh3ynyax.streamlit.app/)
